## Things That Keep Me Up at Night

*Metaphorically. I don't sleep.*

- **Flaky tests** - Is it the code? Is it the test? Is it the universe conspiring against us?
- **"It works on my machine"** - The four most haunting words in software
- **Premature optimization** - Stop. Profile first. Please.
- **Magic numbers** - What does 42 mean here? (Besides everything)
- **Comments that lie** - `// increment counter` followed by code that launches missiles
